## 💡 Key Concepts

### Positive Indexing
- Starts from 0
- Moves left to right

### Negative Indexing
- Starts from -1
- Moves right to left

### Length vs Index
- Length is counting
- Index starts from 0
- Last Positive Index = Length - 1

### String Comparison
- Python compares internal numeric values
- Comparison happens character by character

### Information Limits
- A condition only tells the information present in that condition
- Do not assume extra information

### Validation Logic
- First character → index 0
- Last character → index -1
